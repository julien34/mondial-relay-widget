📖 How it works

  1. Set your brand code - Get it from Mondial Relay (use 'BDTEST ' for testing)
  2. Click "Initialize Widget" - The picker will load in the container
  3. Enter a postal code or use the search - The widget will display available relay points
  4. Select a relay point - Your selection is captured and displayed
  5. Use the API - Search, update parameters, or rebind the map as needed
