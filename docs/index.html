📖 Comment ça marche ?

  1. Configurez votre code brand - Obtenu auprès de Mondial Relay (utilisez 'BDTEST ' pour tester)
  2. Cliquez sur "Initialiser" - Le sélecteur se charge dans le conteneur
  3. Entrez un code postal ou utilisez la recherche - Le widget affiche les points relais disponibles
  4. Sélectionnez un point relais - Votre sélection s'affiche automatiquement
  5. Utilisez l'API - Recherchez, mettez à jour les paramètres ou rafraîchissez la carte
